# Video Streaming Platform

A comprehensive video streaming application that provides users with high-quality video playback and content management capabilities.

## Features

- HD video streaming
- Custom video player controls
- Playlist management
- Video upload and processing
- User authentication
- Content categorization

## Technologies Used

- React
- Video.js
- Node.js
- Express
- FFmpeg for video processing
- AWS S3 for storage
